Download Ubuntu 16.04 LTS Melalui Mirror Lokal Indonesia

Posted on 2016-04-25 | Post modified 2017-06-21 | In Linux | 0 comments
Reading time 3

Assalamualaikum Wr. Wb. Ubuntu 16.04 LTS Xenial Xerus telah dirilis sejak tanggal 21 April 2016 yang lalu. Tentu benyak dari pengguna Ubuntu, atau bahkan orang-orang yang belum pernah menggunakan Ubuntu ingin untuk download Ubuntu 16.04 LTS ini dan mencoba untuk memakainya. Nah, kali ini saya akan berbagi sedikit mengenai mirror lokal Indonesia untuk download Ubuntu 16.04 LTS. Semoga saja dapat mempermudah anda untuk mendapatkan ISO Images dari Ubuntu 16.04 LTS Xenial Xerus ini.

Bagi anda yang ingin download Ubuntu 16.04 ini, saya sarankan apabila anda berada di wilayah Indonesia untuk download melaui mirror lokal yang tersedia. Ada beberapa alasan untuk hal tersebut, salah satunya adalah pada kecepatan koneksi. Jika anda mengunduh melalui Mirror Lokal Indonesia yang tentu saja jaraknya lebih dekat dengan server atau mirror ain yang ada luar negeri. Karenanya kecepatan transfer data saat download Ubuntu 16.04 LTS akan menjadi lebih cepat.

Terdapat begitu banyak Mirror Lokal yang ada di Indonesia, tersebar di berbagai kota, dan biasanya dikelola oleh komunitas open source atau oleh beberapa universitas yang ada di Indonesia.

Dari beberapa data yang saya kumpulkan sudah ada 6 mirror lokal yang menyediakan ISO Images dari Ubuntu 16.04 LTS Xenial Xerus. Mirror tersebut ada di beberapa kota antara lain Jakarta, Bandung, Surabaya, dan Yogyakarta. Anda dapat memilih salah satu yang lokasinya terdekat dengan tempat tinggal anda.

Daftar Mirror Lokal untuk Download Ubuntu 16.04 LTS

  • Jakarta :

    • http://kambing.ui.ac.id/iso/ubuntu/releases/xenial/

  • Bandung :

    • ftp://ftp.itb.ac.id/pub/ubuntu-releases/xenial/

  • Surabaya :

    • http://suro.ubaya.ac.id/pub/ubuntu/16.04/
    • http://buaya.klas.or.id/iso/ubuntu/releases/xenial/
    • http://kebo.pens.ac.id/ubuntu-cd/releases/16.04/

  • Yogyakarta:

    • http://repo.ugm.ac.id/iso/ubuntu/releases/16.04/

Itu dia daftar dari beberapa Mirror lokal yang menyediakan ISO Images dari Ubuntu 16.04 LTS, daftar ini kedepannya akan saya update apabila ada repo lain yang saya temukan mulai menyediakan ISO Images Ubuntu 16.04 LTS.

Sebagai informasi tambahan, pada mirror tersebut anda juga dapat menemukan ISO Images dari distro Linux lain dan juga repository lokal untuk beberapa distro Linux jika dibutuhkan.

Sekian sedikit informasi mengenai daftar mirror lokal Indonesia untuk download ubuntu 16.04 LTS. Semoga informasi ini dapat bermanfaat bagi kita semua. Apabila anda punya tambahan, pertanyaan, kritik, atau saran, silakan sampaikan melalui kolom komentar di bawah.
