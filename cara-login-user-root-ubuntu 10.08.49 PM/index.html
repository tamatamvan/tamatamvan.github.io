Sistem operasi ini banyak digunakan pada berbagai perangkat seperti PC, Smartphone, dan juga tablet. Selain itu Ubuntu juga tersedia untuk penggunaan Server. Seringkali sebagai pengguna Ubuntu baik itu PC ataupun Server kita perlu mengakses sistem melalui user root untuk melakukan modifikasi atau konfigurasi pada sistemnya. Akses root bisa dilakukan menggunakan perintah _sudo _pada terminal. Adakalanya juga kita perlu mengakses sistem dengan user root sesungguhnya. Untuk itu kali ini kita akan mempelajari cara login user root pada ubuntu.

Cara Login User Root pada Ubuntu

Sebelumnya, mungkin setiap kali anda mencoba login sebagai user root pada Ubuntu, sering kali gagal. Begitu juga ketika mengetikkan perintah su dan memasukkan password, upaya tersebut gagal dan menunjukkan pesan su: Authentication failure.

[caption id=‚Äùattachment_416‚Äù align=‚Äùaligncenter‚Äù width=‚Äù398‚Äù] User Root Authentication Failure[/caption]

Sebenarnya untuk untuk bisa login sebagai user root pada Ubuntu, yang perlu kita lakukan hanyalah menambahkan password pada _user root. _Untuk menambahkan password pada _user root di Ubuntu, kita hanya perlu mengetikkan perintah sudo passwd root. _Setelah itu anda akan diminta memasukkan password user yang anda gunakan saat ini, baru setelah itu anda diminta untuk memasukkan password root yang anda inginkan. Setelah itu akan muncul informasi _"passwd: password updated successfully" _yang berarti password root anda telah berhasil diperbarui.

[caption id=‚Äùattachment_414‚Äù align=‚Äùaligncenter‚Äù width=‚Äù383‚Äù] Pemberian Password User Root[/caption]

Setelah password root anda berhasil diperbarui, adan dapat mencoba login sebagai user root di _terminal dengan perintah **su**, setelah itu masukkan password root anda. Jika berhasil tanda $ pada terminal anda akan berubah menjadi tanda #_, dan username yang tercantum pada terminal juga akan berubah menjadi _root._
