Mengganti Port Server SSH Di Linux

Posted on 2016-02-09 | Post modified 2017-06-21 | In Linux | 0 comments
Reading time 2

Assalamualaikum Wr. Wb., Kali ini saya akan membagikan tutorial tentang cara mengganti port server SSH di Linux. Tutorial kali bisa diterapkan pada berbagai distro Linux, baik itu Ubuntu, Debian, Fedora, CentOS, atau distro lainnya. Mengganti port server SSH di Linux dari port ssh default (22) biasanya dilakukan karena beberapa alasan, di antaranya seperti untuk mengurangi angka kegagalan login, dan juga meningkatkan keamanan dari serangan terhadap server melalui SSH.

Mengganti Port Server SSH di Linux

Berikut adalah cara yang dapat kita lakukan untuk mengganti port server SSH di Linux :

  1. Login menggunakan user root, atau tambahkan _**sudo**_pada setiap awal perintah.

  2. Lakukan backup terhadap file konfigurasi sshd dengan mengetikkan perintah berikut

    _cp /etc/ssh/sshd_config /etc/ssh/sshd_config_backup_`

  3. Selanjutnya ketikkan perintah untuk mengedit file configurasi sshd_configmenggunakan teks editor yang anda gunakan, di sini saya menggunakan teks editor nano, ketikkan perintah berikut :
    _**`nano /etc/ssh/sshd_config`**_
  4. Pada line atau baris ke-5 pada file _**sshd_config**_ anda akan menemukan tulisan Port 22, itu merupakan daftar port yang di dengarkan oleh server SSH kita, dengan kata lain melalui port tersebutlah server SSH dapat diakses. Tulisan _**Port 22 **_itulah yang akan kita ubah sesuai dengan keinginan kita, misalnya dari _**Port 22**_kita ubah menjadi Port 1734, atau nomor port lainnya.

  5. Selain mengganti port, anda juga dapat menambahkan beberapa port kedalam file konfigurasi tersebut dengan menambahkan tulisan _Port [nomor_port]_ setelah port default, sehingga server SSH dapat diakses melalui lebih dari satu port.

  6. Selanjutnya restart SSH dengan perintah :

    _**service restart sshd**_

    apabila gagal, coba gunakan perintah :
    `service restart ssh

  7. Selesai, setelah itu anda bisa melakukan pengujian dengan mengakses server anda melalui port yang anda cantumkan di file _sshd_config_ dengan menggunakan Putty, WinSCP, atau software lain sejenisnya.
