Hal Yang Perlu Dilakukan Setelah Install Ubuntu 16.04 LTS

Posted on 2016-05-05 | Post modified 2017-06-21 | In Linux | 0 comments
Reading time 13

Assalamu‚Äôalikum Wr. Wb. Bagi anda yang baru saja selesa menginstall Ubuntu 16.04 LTS Xenial Xerus mungkin masih mencari tahu tentang hal yang perlu dilakukan setelah Install Ubuntu 16.04 LTS. Memang pada dasarnya setelah menginstall Ubuntu 16.04 LTS masih ada beberapa hal yang perlu kita lakukan untuk menjadikan sistem operasi Ubuntu 16.04 LTS milik kita menjadi benar-benar siap pakai untuk berbagai keperluan kita, dan semua itu bersifat relatif tergantung pada kebutuhan kita. Akan tetapi, berikut saya jelaskan beberapa hal yang mungkin anda perlukan untuk menjadikan Ubuntu 16.04 LTS menjadi Ubuntu yang anda butuhkan dan inginkan.

Hal Yang Perlu Dilakukan Setelah Install Ubuntu 16.04 LTS Xenial Xerus

  1. Telusuri segala hal yang baru dari Ubuntu 16.04 LTS Xenial Xerus Ada baiknya bagi anda jika belum mengetahui apa saja yang baru dari rilis Ubuntu terbaru ini untuk menelusuri apa saja yang baru dari Ubuntu 16.04 LTS ini. Anda dapat membaca artikel yang saya baca sebelumnya mengenai Fitur Baru Ubuntu 16.04 LTS Xenial Xerus untuk mengetahui berbagai fitur barunya, atau juga membaca artikel dari sumber lain yang terkait.

  2. Check Update Terakhir yang Tersedia Menurut saya ini merupakan hal yang paling penting untuk dilakukan setelah install Ubuntu 16.04 LTS Xenial Xerus. Penting bagi kita untuk menjaga sistem yang kita gunakan tetap up-to-date untuk menjaga performa dan keamanannya. Selain untuk menginstall beberapa aplikasi yang kita butuhkan pertama kali setelah install Ubuntu 16.04 LTS kita memang harus melakukan check update atau update reposiroty terlebih dahulu.Untuk melakukan check update pada Ubuntu 16.04 LTS dapat kita lakukan dengan cara menjalankan perintah sudo apt update, setelah itu apabila terdapat update yang tersedia dapat kita install dengan menjalankan perintah sudo apt dist-upgrade.

    [caption id=‚Äùattachment_635‚Äù align=‚Äùaligncenter‚Äù width=‚Äù697‚Äù] Software Updater pada Unity Dash[/caption]

    Selain menggunakan command line pada terminal, kita juga dapat melakukkannya dengan menjalankan ‚ÄòSoftware Updater‚Äô. Cukup ketikkan ‚Äòupdate‚Äô pada unity dash, dan pilih ‚ÄòSoftware Updater‚Äô untuk dijalankan. Secara otomatis software updater akan mengecheck update yang tersedia, setelah itu kita akan dimintai konfirmasi untuk menginstallnya atau tidak.

    [caption id=‚Äùattachment_636‚Äù align=‚Äùaligncenter‚Äù width=‚Äù481‚Äù] Software Updater Berjalan[/caption]

  3. Install Codec Multimedia Ubuntu (Ubuntu Restricted Extras) ‚ÄúUbuntu restricted extras‚Äù adalah sebuah paket yang terdiri dari beberapa codec yang tidak dimasukkan pada system Ubuntu secara default karena masalah legalitas pada beberapa negara.Untuk menghindari masalah legalitas, Ubuntu tidak memasukkan beberapa codec tersebut secara default dalam sistem mereka. Akan tetapi pihak Ubuntu menyediakan pake Ubuntu Restricted Extras ini secara terpisah sehingga pengguna dapa menginstallnya dengan mudah apabila diperlukan.Paket ini aman untuk diinstall, setelah menginstallnya kita akan dapat memutar berbagai format multimedia seperti MP3, MP4, AVI, MKV, dan lain sebagainya. Untuk menginstall Ubuntu Restricted Extras, cukup jalankan perintah sudo apt install ubuntu-restricted-extraspada terminal kita.

  4. Install Propietary Driver Ubuntu mendukung sebagian besar Graphic Card Nvidia dan Intel dengan memberi anda opsi untuk menggunakan driver open source atau driver propietary yang dikeluarkan perusahaan hardware masing-masing.Ada baiknya untuk mendapatkan performa terbak dar sistem Ubuntu anda, anda dapat menggunaka driver propietary yang tersedia untuk perangkat atau komputer yang anda gunakan.Hal ini akan jadi cukup penting terutama apabila anda berniat menjalankan aplikasi yang membutuhkan performa grafis yang baik seperti Blender, atau bermain game dengan Steam.

    [caption id=‚Äùattachment_638‚Äù align=‚Äùaligncenter‚Äù width=‚Äù906‚Äù] Install Propietary Additional Drivers[/caption]

    Untuk menginstall propietary driver yang tersedia silakan ikuti langkah-langkah berikut ini :

    1. Buka dan jalankan ‚ÄòSoftware & Updates‚Äô melalui Unity Dash
    2. Klik pada tab ‚ÄòAdditional Drivers‚Äô
    3. Klik, ceklist pada driver yang ingin anda install, ikuti setiap instruksi yang muncul, kemudian klik ‚Äúinstall and apply any changes‚Äù

  5. Install Unity Tweak Unity Tweak merupakan sebuah software yang memungkinkan anda untuk mengonfigurasi Unity seperti memindahkan posisi Unity Launcher, mengatur posisi menu bar, mengubah ukuran icon pada Unity Launcher, mengganti tema dan ikon, dll dengan mudah tanpa harus menjalankan perintah melalui command line.Unity Tweak tersedia pada reposiroty Ubuntu, untuk menginstallnya anda dapat mencarinya melalui Ubuntu Software atau mengetikkan perintah sudo apt install unity-tweak.

    [caption id=‚Äùattachment_640‚Äù align=‚Äùaligncenter‚Äù width=‚Äù703‚Äù] Unity Tweak Tool[/caption]

  6. Menentukan Posisi Unity Launcher Setelah selesai menginstall Unity Tweak Tool, kita dapat mengonfigurasi berbagai pengaturan pada Desktop Unity kita. Pada Ubuntu 16.04 LTS ini kita dapat mengubah posisi Unity Launcher pada sisi kiri layar atau sisi bawah layar desktop kita. Untuk menentukan posisi tersebut dapat kita lakukan melalui Unity Tweak Tool dengan pada bagian _U**nity > Launcher > Posistion, lalu pilih ‚ÄòLeft‚Äô atau ‚ÄòBottom‚Äô.
[caption id="attachment_641" align="aligncenter" width="500"][![Memindah Posisi Unity Launcher - hal yang perlu dilakukan setelah install ubuntu 16.04](https://tamatamvan.web.id/wp-content/uploads/2016/04/2016-04-06-20_25_32.gif)](https://tamatamvan.web.id/wp-content/uploads/2016/04/2016-04-06-20_25_32.gif) Memindah Posisi Unity Launcher[/caption]

Bisa juga dengan mengetikkan perintah :


**`gsettings set com.canonical.Unity.Launcher launcher-position Bottom`**
7. **Mengaktifkan fitur Minimize on Click pada Unity Launcher **Secara default, ketika kita klik pada salah satu icon pada Unity Laucher akan membuka atau menjalankan aplikasi yang kita klik icon-nya tersebut. Akan tetapi saat kita tidak bisa me-minimize aplikasi yang baru saja kita jalankan dengan klik kembali pada icon yang sama. [caption id="attachment_642" align="aligncenter" width="516"][![Minimize on Click Unity Launcher - hal yang perlu dilakukan setelah install ubuntu 16.04](https://tamatamvan.web.id/wp-content/uploads/2016/04/unity-launcher.gif)](https://tamatamvan.web.id/wp-content/uploads/2016/04/unity-launcher.gif) Minimize on Click Unity Launcher[/caption]  
Untuk melakukkannya kita perlu mengaktifkan fitur Minimize on Click pada Unity Laucher melalui Unity Tweak Tool pada bagian **_Unity > Launcher > Minimize single window application on click_**. Atau anda juga dapat mengaktifkannya dengan menjalankan perintah : 
**`gsettings set org.compiz.unityshell:/org/compiz/profiles/unity/plugins/unityshell/ launcher-minimize-window true`**
8. **Tentukan posisi app menu muncul **Secara default, app menu atau menu bar pada Ubuntu berapa pada top panel Unity dan tersembunyi secara otomatis setelah beberapa detik. Menu ini akan muncul jika anda mengarahkan cursor mouse anda ke atasnya. [caption id="attachment_643" align="aligncenter" width="512"][![Posisi App Menu - hal yang perlu dilakukan setelah install ubuntu 16.04](https://tamatamvan.web.id/wp-content/uploads/2016/04/Screenshot-from-2016-04-28-15-42-46-e1461833162813.png)](https://tamatamvan.web.id/wp-content/uploads/2016/04/Screenshot-from-2016-04-28-15-42-46-e1461833162813.png) Posisi App Menu[/caption] Ubuntu menyediakan pilihan pada _System Settings > Appearance_ yang memungkinkan anda untuk menentukan dimana posisi dari App Menu atau menu bar ini akan muncul.Jika anda ingin menu bar tersebut muncul pada window aplikasi yang anda jalankan, silakan ikuti langkah berikut ini: 1. **Buka ‚ÄòSystem Settings > Appearance‚Äô** 2. **Pilih tab ‚ÄòBehavior‚Äô** 3. **Temukan bagian berjudul: ‚ÄòShow the Menus for a Window‚Äô** 4. **Checklist (click) lingkaran disamping ‚ÄòIn the window‚Äôs title bar‚Äô** Anda juga dapat membuat menu bar selalu tampak dengan mengikuti langkah dibawah ini: 1. **Buka ‚ÄòSystem Settings > Appearance‚Äô** 2. **Pilih tab 'Behaviour'** 3. **Temukan bagian berjudul: ‚ÄòMenus visibility‚Äô** 4. **Checklist (click) lingkaran disamping ‚ÄòAlways displayed‚Äô** 9. **Install berbagai aplikasi yang anda butuhkan** Tidak semua aplikasi yang anda butuhkan terisntall secara default pada sistem Ubuntu dan juga tersedia pada Ubuntu Software. Faktanya begitu banyak aplikasi linux yang bagus berada diluar repository utama Ubuntu. Anda dapat menginstall beberapa aplikasi tersebut dengan cara mengunduh file installer .deb nya sendiri.Untuk menginstall paket berekstensi .deb, anda dapat mengikuti tutorial yang sudah pernah saya tulis sebelumnya, [di sini](https://tamatamvan.web.id/linux/cara-install-paket-deb-ubuntu/).Selain itu, berikut daftar aplikasi yang saya rekomendasikan untuk anda install pada Ubuntu 16.04 LTS anda : * **[Google Chrome](http://google.com/chrome)** ‚Äì Web browser with built-in, up-to-date Flash for Linux * **[Nylas N1](https://nylas.com/n1) **‚Äì A modern, stylish email client * **[Dropbox](https://www.dropbox.com/install?os=lnx)** ‚Äì Cloud storage service * **[Spotify for Linux](http://spotify.com)** ‚Äì Music streaming service * **[Lightworks Free](https://www.lwks.com/index.php?option=com_lwks&view=download&Itemid=206&tab=1) ‚Äì** Professional non-linear video editor * **[Viber](http://www.viber.com/en/)** ‚Äì Cross-platform Skype alternative * **[Vivaldi](https://vivaldi.net/en-US/)** ‚Äì Developer focused web browser * [**Xtreme Download Manager** ](http://xdman.sourceforge.net/)- Download Manager Open Source, sejenis IDM untuk linux. * **[Steam](http://store.steampowered.com/about/) - **Game distributors, install, manage, and play a lot of games. 10. **Kustomisasi tampilan Ubuntu 16.04 LTS sesuka hati Anda **Jika anda merasa kurang suka dengan tampilan default Ubuntu 16.04 LTS atau ingin suasana yang lebih _fresh _pada tampilan desktop anda, anda bisa mengubah tampilan desktop Ubuntu 16.04 LTS sesuka hati anda. Anda bisa mengganti icon, GTK Themes, Install aplikasi widget seperti Conky, atau bahkan menginstall Desktop Environment lain seperti XFCE, LXQt, Cinnamon, dsb. [caption id="attachment_647" align="aligncenter" width="1024"][![Tampilan Ubuntu yang sudah dikustomisasi - Hal yang perlu dilakukan setelah install ubuntu 16.04 LTS](https://tamatamvan.web.id/wp-content/uploads/2016/05/Screenshot-from-2016-04-23-20-40-35-1024x576.jpg)](https://tamatamvan.web.id/wp-content/uploads/2016/05/Screenshot-from-2016-04-23-20-40-35.jpg) Tampilan Ubuntu yang sudah dikustomisasi[/caption] Untuk mengkustomisasi tampilan Ubuntu 16.04 LTS seperti mengubah icon atau GTK Themes, saya merekomendasikan situs [Noobslab.com](http://www.noobslab.com/). Situs tersebut menyediakan berbagai Icon pack, GTK Themes, Conky, beserta tutorial cara instalasinya. Selain menyediakan berbagai icon packs, GTK Themes, Conky, dan berbagai macam hal untuk kustomisasi tampilan, pada situs tersebut juga terdapat info mengenai berbagai software menarik yang mungkin dapat meningkatkan produktifitas anda dengan Ubuntu. Sepuluh hal diatas adalah beberapa hal yang perlu dilakukan setelah install Ubuntu 16.04 LTS yang saya anggap sangat penting. Selain sepuluh hal diatas juga masih banyak hal yang bersifat optional menurut saya, akan tetapi bisa anda lakukan untuk meningkatkan performa Ubuntu anda seperti : 1. _**Install aplikasi pembersih sistem BleachBit** _(semacam CCleaner pada windows), cukup dengan ketikkan perintah : 
**`sudo apt install bleachbit` **

  1. Install dan lakukan syncronitation dengan layanan cloud storage yang anda gunakan pada Ubuntu. Berikut ada beberapa daftar layanan cloud storage yang bisa anda gunakan pada Ubuntu :

    • Dropbox menyediakan aplikasi client Linux yang terupdate secara regular.
    • OwnCloud layanan ini memungkinan anda membuat personal cloud drive anda sendiri yang dapat anda kontrol sepenuhnya.
    • OneDrive dapat diakses melalui third-party scripts
    • InSync menawarkan dukungan unofficial untuk Google Drive Linux (gratis)
    • MEGA menawarkan ruang penyimpanan gratis yang cukup besar, yaitu sebesar 50GB.Dari daftar tersebut, saya sendiri menggunakan Dropbox dan MEGA, dan yang saya rasakan performanya sejauh ini untuk keduanya cukup memuaskan, silakan anda pilih layanan cloud storage yang paling tepat untuk anda.

  2. Integrasikan Online Accounts yang anda miliki dengan Ubuntu 16.04 LTS. Anda dapat melakukannya dengan mengakses Settings > Online Accounts dan tambahkan account yang anda inginkan.

    [caption id=‚Äùattachment_645‚Äù align=‚Äùaligncenter‚Äù width=‚Äù912‚Äù] Online Accounts Integration[/caption]

     

  3. Install Laptop Mode Tools. Jika anda menggunakan Ubuntu 16.04 LTS pada laptop, ada baiknya anda menginstall laptop mode tools untuk meningkatkan efisiensi konsumsi daya baterai pada laptop anda, sehingga usia dan kualitas baterai laptop akan menjadi lebih terjaga. Untuk menginstallnya cukup ketikkan perintah :

    **`sudo apt install laptop-mode-tools`**

  4. Install Java._ _Banyak aplikasi open source pada linux berjalan dengan menggunakan Java. Terdapat beberapa versi Java yang anda gunakan pada Ubuntu 16.04 LTS, seperti OpenJDK, atau Oracle Java. Saya sendiri merekonmendasikan Oracle Java. Untuk menginstallnya anda dapat mengikuti tutorial Install Oracle Java pada Ubuntu.
