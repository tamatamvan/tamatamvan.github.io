Reading time 3

Assalamu‚Äôalaikum Wr. Wb., Akses remote SSH server biasanya dilakukan untuk mengontrol atau mengendalikan server dari kejauhan. Biasanya kita melakukan remote server via SSH dengan dua tipe, yaitu remote console (menggunakan putty, atau Terminal) atau menggunakan SSH File Transfer Protocol (SFTP) yang biasanya menggunakan software WinSCP dan FIleZilla. Pada umumnya banyak orang menggunakan WinSCP untuk remote server SSH dengan SFTP bagi pengguna Windows. Namun, untuk pengguna Liinux bisa menggunakan software alternatif seperti FileZilla pada Linux. Untuk itulah kali ini saya akan berbagi tentang akses SSH server dengan FileZilla

FileZilla adalah sebuah software Client FTP opensource yang berlisensi GNU GPL. Software ini memiliki performa yang cukup bagus untuk keperluan transfer file. Selain itu FileZilla juga dapat digunakan untuk akses SSH File Transfer Protocol (SFTP). Bagi anda yang belum memiliki software FileZilla dapat mendownload FileZilla di sini.
