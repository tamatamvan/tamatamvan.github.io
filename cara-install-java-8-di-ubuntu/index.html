Cara Mudah Install Oracle Java 8 Pada Ubuntu Atau Linux Mint

Reading time 2

Kali ini saya akan berbagi mengenai bagaimana cara mudah untuk menginstall program Oracle Java 8 pada sistem operasi Ubuntu/Linux Mint atau sistem operasi berbasis Ubuntu lainnya. Pada tutorial kali ini, kita akan mempelajari mengenai cara menginstall Oracle Java 8 melalui repository ppa yang disediakan oleh WEB UPD8.

Sebenarnya terdapat alternatif lain dimana kita bisa menginstall OpenJDK untuk menggunakan Java tanpa harus menambahkan ppa untuk menginstall oracle java. Akan tetapi dalam beberapa kasus saya pernah mengalami beberapa program yang berjalan di Java mengalami beberapa masalah saat berjalan dengan OpenJDK, salah satunya ialah Android Studio yang not responding ketika akan memilih API untuk sebuah project baru.

Apabila sebelum menginstall Oracle Java 8 pada sistem anda telah terinstall OpenJDK, anda tidak perlu menguninstall OpenJDK tersebut. Dalam packet yang akan kita install pada tutorial ini meliputi JDK, JRE, dan Java Web Plugin, serta mendukung baik 32bit atau 64bit. Namun, semua itu akan terinstall secara otomatis karena kita tidak bisa memilih satu persatu.

Cara Mudah Install Oracle Java 8 pada Ubuntu atau Linux Mint :

  1. Buka terminal anda dan masukkan perintah

    sudo add-apt-repository ppa:webupd8team/java`
    setelah itu, tunggu beberapa saat, lalu tekan enter untuk mengonfirmasi.

  2. Selanjutnya update repository Ubuntu atau Linux Mint anda menggunakan perintah
    `sudo apt-get update`
  3. Setelah proses update repository selesai, ketikkan perintah
    `sudo apt-get install oracle-java8-installer`
  4. Tunggu hingga proses download dan instalasi selesai dengan otomatis. Untuk memastikan Oracle Java 8 telah terinstall ketikkan perintah
    `java -version`
  5. Apabila Oracle Java 8 telah benar-benar terinstall, maka akan muncul tampilan seperti dibawah ini :
    `java version ‚Äú1.8.0_66‚Äù
    Java(TM) SE Runtime Environment (build 1.8.0_66-b17)
    Java HotSpot(TM) 64-Bit Server VM (build 25.66-b17, mixed mode)
