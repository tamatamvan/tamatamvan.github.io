Untuk redirect atau mengalihkan halaman web sendiri kita dapat melakukannya dengan banyak cara, seperti menggunakan .htaccess seperti pada tutorial yang pernah saya tulis sebelumnya, menggunakan PHP, atau menggunakna JavaScript sepeti yang akan kita bahas saat ini.

Redirect halaman web dengan JavaScript sendiri kita dapat menggunakan beberapa variasi, seperti contohnya menggunakan tombol atau button yang akan mengeksekusi fungsi redirect, atau redirect secara otomatis setelah halaman dimuat, redirect otomatis setelah beberapa detik.

Semua fungsi redirect tersebut dapat kita kustomisasi sesuai dengan keperluan dan kebutuhan kita masing-masing. Berikut ini adalah contoh penerapan page redirection atau mengalihkan halaman web dengan JavaScript.
