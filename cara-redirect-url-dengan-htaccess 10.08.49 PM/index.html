Cara Redirect URL dengan .htaccess

Ada beberapa langkah yang harus kita lakukan untuk melakukan redirecting atau pengalihan URL dengan .htaccess. Cara redirect url dengan .htaccess adalah sebagai berikut :

  1. Pastikan website atau aplikasi web anda berjalan pada web server apache, hal ini sangatlah penting dikarenakan tidak semua web server mendukung penggunaan file .htaccess. Jika anda tidak yakin mengenai hal ini, anda dapat menghubungi penyedia layanan hosting anda untuk menanyakannya.
  2. Periksa root directory dari website anda (folder tempat semua file website atau aplikasi web anda disimpan), pastikan tidak ada file yang disembunyikan, check apakah ada file .htaccess atau tidak, apabila ada silakan lakukan pengeditan terhadap file tersebut.
  3. Buat file .htaccess. Apabila anda tidak menemukan file .htaccess, anda dapat membuat file .htaccess yang yang baru menggunakan aplikasi text editor seperti notepad atau yang lainnya. Pastikan file anda simpan dengan nama ‚Äú.htaccess‚Äú, jangan lupakan tanda titik (.) pada awal nama file. Pastikan juga file tersebut tidak memiliki ekstensi di belakang nama file-nya seperti .txt.

  4. Ketikkan code .htaccess yang akan kita gunakan untuk melakukan redirecting. Ketikkan format code berikut pada file .htaccess yang akan kita buat: redirect 301 /old/url_lama.com http://www.url_baru.com

    • Ganti bagian ‚Äúurl_lama.com‚Äú dengan link atau URL yang ingin anda alihkan, pastikan anda tidak menambahkan ‚Äúhttp://www. ‚Äú pada bagian setelah ‚Äú/old/‚Äú.
    • Ganti bagian ‚Äúurl_baru.com‚Äù dengan link atau URL baru anda.
    • Pastikan antara bagian ‚Äú/old/url_lama.com‚Äù dengan ‚Äúhttp://www.url_baru.com‚Äú terdapat satu spasi.
    • Contoh, apabila saya ingin mengalihkan pengunjung [tamatamvan.xyz](http://tamatamvan.xyz) menuju [tamatamvan.web.id](http://tamatamvan.web.id)maka code yang saya ketikkan pada file .htaccess saya adalah sebagai berikut redirect 301 /old/tamatamvan.xyz http://tamatamvan.web.idatau redirect 301 /old/tamatamvan.xyz http://www.tamatamvan.web.id, terserah pada kita apabila ingin mengalihkan link ke URL baru yang menggunakan ‚Äúwww‚Äú atau tidak. Namun sekali lagi perlu diingat, untuk tidak menambahkan ‚Äúhttp://‚Äú atau ‚Äúwww‚Äù pada URL lama atau URL perujuk.

  5. Simpan atau upload file .htaccess yang telah kita buat kedalam root directory web server kita. Pastikan sebelumnya kita telah melakukan backup terhadap file .htaccess yang lama jika ada.

  6. Terakhir, lakukan pengujian dengan membuka link lama yang ingin kita alihkan. Ada baiknya bagi kita untuk melakukan pengujian dengan menggunakan mode Incognito atau _Private Browsing _untuk memastikan web browser anda langsung mengakses file redirect yang baru daripada mengakses cache yang ada pada web browser. Atau anda bisa menghapus semua cache yang ada pada browser anda melalui pengaturan web browser anda.
  7. Selamat mencoba, dan semoga berhasil!
