Menambahkan User Account Baru Ubuntu

Posted on 2016-04-09 | Post modified 2017-06-21 | In Linux | 0 comments
Reading time 4

Assalamualaikum Wr. Wb. Ubuntu merupakan salah satu distro Linux yang paling populer, bahkan dari kepopuleran Ubuntu menjadikan banyak distro baru yang merupakan turunannya bermunculan. Hal ini tidak lain karena sifat Ubuntu yang relatif user friendly sehingga cocok untuk pengguna Linux pemula. Kali ini saya akan membahas sebuah tutorial sederhana untuk pemula Ubuntu, yaitu sebuah tutorial tentang cara menambahkan user account baru Ubuntu.

Penggunaan lebih dari satu user account terkadang diperlukan ketika sebuah komputer atau PC digunakan oleh lebih dari satu orang. Dengan adanya lebih dari satu user, biasanya setiap user atau pengguna memiliki karakter masing-masing, dan keperluan yang berbeda dalam penggunaan komputer. Oleh sebab itu terkadang diperlukan adanya lebih dari satu user account pada sistem operasi yang kita jalankan pada komputer kita, sehingga setiap user dapat mengatur data, aplikasi, dan segala keperluan pekerjaan-nya masing-masing dengan komputer tersebut dengan lebih leluasa.

Bagi anda yang baru saja menggunakan sistem operasi Ubuntu dan ingin menambahkan user account baru Ubuntu pada sistem anda, anda dapat mengikuti tutorial di bawah ini.
