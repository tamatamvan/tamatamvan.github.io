Seberapa Banyak Dan Dimana Saja Penggunaan Ubuntu

Posted on 2016-04-12 | Post modified 2017-06-21 | In Linux , Open Source Software | 0 comments
Reading time 3

Assalamu‚Äôalaikum Wr. Wb. Ubuntu merupakan salah satu distro Linux yang paling populer saat ini. Distro yang pertama kali rilis pada tahun 2004 kini telah menjadi salah satu distro terbesar Linux dan juga telah menurunkan begitu banyak distro lain seperti Linux Mint, ZorinOS, Deepin, dll. Ubuntu juga mengalami perkembangan dari waktu ke waktu, hingga saat ini ada berbagai varian Ubuntu termasuk Ubuntu Server, dan Ubuntu Phone untuk tablet PC dan smartphone. Dan kali ini kita akan membahas mengenai penggunaan Ubuntu, di mana saja Ubuntu digunakan, dan seberapa banyak Ubuntu digunakan.

Berapa banyak orang menggunakan Ubuntu? Ada sangat banyak!

Cannonical membagikan beberapa statistik yang menarik dalam sebuah infografik (yang akan anda lihat di bawah) yang menawarkan wawasan lebih jauh kepada anda tetang seberapa banyak dan dimana saja penggunaan Ubuntu ada.

Satu cloud instace Ubuntu diluncurkan setiap detiknya.

Ubuntu digunakan di berbagai tempat, dari stasiun luar angkasa dan kendaraan Mars rovers, hingga robot dan ruter, juga server websites, dan computer seperti yang anda gunakan.

Penggunaan Ubuntu benar-benar ada di mana saja.

Ubuntu sebenarnya begitu banyak digunakan oleh orang-orang. Begitu banyak orang menggunakan dan mengakses Ubuntu baik secara langsung atau tidak langsung, bahkan tanpa mereka sadari!

Ubuntu berjalan _di belakang layar di _Walmart dan Netflix dan membantu proses selfie pada Snapchar dan juga entri data pada Instagram. Ubuntu menjadi otak dari supercomputer paling powrful di dunia dan menjadi otot di balik ensiklopedia online terbesar.

Akan tetapi mendeteksi dan mencari tahu seberapa banyak orang menggunakan Ubuntu bukanlah hal yang mudah, sebagai mana Cannonical, perusahaan di balik Ubuntu mengakui :

‚ÄúPengguna tidak perlu mendaftarkan instalasi mereka, berbagai perangkat pre-load dengan software dan server yang ditenagai dengan Ubuntu.‚Äù

Akan tetapi statistik mereka dapat menunjukkan jejak global, dengan ratusan juta PC, server, dan bergai perangkat, virtual machine, dan container memanfaat kan open-source OS (Ubuntu).

Dan bukan tidak mungkin penggunaan Ubuntu akan semakin meningkat, mengingat perkembangan Ubuntu yang semakin mencakup berbagai penggunaan, yang kini bahkan juga telah hadir Ubuntu Phone untuk smartphone, dimana kita bahkan bisa menginstallnya sendiri pada perangkat smartphone atau tablet Google Nexus. Selain itu teknologi cloud computing dan IoT (Internet of Things) juga semakin mendukung perkembangan Ubuntu.
