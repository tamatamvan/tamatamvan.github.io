Cara Install Laravel Dengan Mudah

Posted on 2016-07-10 | Post modified 2017-06-21 | In Pemrograman , Pemrograman Web | 0 comments
Reading time 3

Assalamu‚Äôalaikum Wr. Wb. Beberapa tahun terakhir Laravel telah menjadi framework PHP yang sangat populer dan semakin banyak dipakai oleh para web developer di seluruh dunia. Dengan segala kelebihan dan berbagai fitur menariknya, Laravel mampu memikat para web developer untuk mempelajarinya, serta menggunakannnya untuk memudahkan proses pengembangan website. Oleh karena itu kali ini saya akan membahas mengenai cara install Laravel dengan mudah.

Pada tutorial kali ini kita akan menggunakan bantuan software Composer PHP dependency manager, apabila anda belum memiliki Composer terinstall pada sistem anda, anda dapat menginstallnya terlebih dahulu. Untuk menginstall composer sendiri, anda dapat mengikuti tutorial berikut ini.

Jika anda sudah menginstall Composer pada sistem anda, anda dapat memulai menginstall laravel dengan mengikuti tutorial di bawah. Oh ya, pastikan juga anda terhubung dengan koneksi internet saat melakukan instalasi laravel.

Cara install Laravel dengan mudah

  1. Setelah anda selesai menginstall Composer silakan anda buka Terminal apabila anda menggunakan Linux atau Mac OSX, atau Command Prompt jika anda menggunakan Windows.
  2. Selanjutnya, ketikkan perintah untuk masuk ke root directory pada web server lokal anda (seperti public_html, /var/www, atau htdocs) dengan menggunakan perintah cd. Contoh, apabila root directory saya berada pada folder Documents/public_html, maka saya ketikkan perintah cd Documents/public_html.
  3. Jika sudah masuk ke root directory pada web server lokal, ketikkan perintah composer untuk menginstall laravel sebagai berikut : composer create<span class="token operator">-</span>project <span class="token operator">--</span>prefer<span class="token operator">-</span>dist laravel<span class="token operator">/</span>laravel nama-folder. (Ket: opsi --prefer-dist menunjukkan versi laravel yang akan kita install adalah versi stabil. Parameter nama-folder adalah folder lokasi setiap file aplikasi laravel akan disimpan. Parameter tersebut namanya dapat anda ubah dan sesuaikan sesuai kebutuhan dan keperluan anda. Sebagai contoh jika akan menginstall laravel pada folder belajar-laravel, perintah yang diketikkan adalah *composer create<span class="token operator">-</span>project <span class="token operator">--</span>prefer<span class="token operator">-</span>dist laravel<span class="token operator">/</span>laravel belajar-laravel)
  4. Setelah itu jalankan perintah tersebut, dan tunggu hingga proses selesai secara otomatis.

Cara akses Laravel setelah instalasi selesai

Setelah instalasi selesai, anda dapat langsung mencoba mengakses hasil instalasi laravel anda melalui browser dengan mengetikkan URL localhost/nama_folder_laravel/public/, jika berhasil akan muncul tulisan ‚ÄúLaravel 5‚Äù dengan tampilan seperti pada gambar di bawah.
