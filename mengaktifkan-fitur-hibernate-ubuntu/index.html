Mengaktifkan Fitur Hibernate Pada Ubuntu

Posted on 2016-01-20 | Post modified 2017-06-21 | In Linux | 0 comments
Reading time 5

Asslamuâ€™alaikum Wr. Wb. Fitur hibernate merupakan sebuah fitur yang cukup bermanfaat dan memudahkan pengguna dalam bekerja dengan komputer. Fitur ini memungkinkan kita untuk mematikan PC atau komputer secara total, akan tetapi dengan tetap menyimpan sesi sebelummnya ketika PC memasuki mode hibernate, sehingga ketika kita menghidupkan PC atau komputer tersebut kembali, ia akan langsung menampilkan kembali atau melanjutkan sesi dari sistem operasi tersebut sebelum masuk ke mode hibernate. Hal ini dimungkinkan karena ketika sebuah PC akan memasuki mode hibernate, sistem operasi akan menyimpan data sesinya, meliputi program-program yang berjalan ke dalam file yang disimpan di harddisk. Selanjutnya, ketika PC yang dalam mode hibernate dihidupkan kembali, sistem operasi akan secara otomatis membaca file data hibernate yang telah disimpan, dan melanjutkan sesi sebelum PC dimatikan (dalam mode hibernate) berdasarkan data dari file tersebut. Kali ini saya akan berbagi mengenai cara mengaktifkan fitur hibernate pada Ubuntu.

Nah, permasalahan yang akan kita bahas kali ini adalah tidak tersedianya secara default fitur hibernate pada sistem operasi Ubuntu. Kita bisa melihat hal tersebut melalui menu yang akan muncul ketika kita klik menu power pada sudut kanan atas layar PC atau komputer kita. Pada menu tersebut secara default hanya akan ditampilkan pilihan : Log Out, Suspend (sleep), dan Shut Down. Selain itu, apabila kita menekan tombol fisik power komputer atau PC kita secara langsung akan muncul tampilan prompt dialog, akan tetapi sekali lagi secara _default _tidak menampilkan pilihan hibernate.

Pilihan untuk melakukan hibernate melalui antartmuka grafis dapat kita munculkan pada sistem operasi Ubuntu, kita hanya perlu mengaktifkan fitur hibernate pada Ubuntu secara manual dengan membuat sebuah file konfigurasi tertentu. Namun, sebelumnya kita juga harus memastikan bahwa terdapat partisi swap yang berukuran paling tidak sama besar dengan kapasitas RAM yang ada pada komputer kita untuk digunakan oleh sistem operasi Ubuntu sebagai tempat penyimpanan data atau _hibernate Ubuntu. _Untuk mengaktifkan fitur hibernate pada Ubuntu silakan ikuti langkah-langkah dibawah ini.

Cara Mengaktifkan Fitur Hibernate pada Ubuntu

  1. Lakukan _test untuk memastikan fitur hibernate dapat bekerja pada komputer atau PC dengan sistem operasi Ubuntu anda dengan mengetikkan perintah sudo pm-hibernate_`pada terminal anda.`
  2. Setelah komputer atau PC anda mati, nyalakan kembali PC anda, apabila aplikasi yang sebelumnya anda jalankan tetap berkerja, itu berarti fitur hibernate bekerja dengan baik. Apabila tidak, silakan cek ketersediaan partisi _swap _anda.
  3. Langkah selanjutnya, buka teminal emulator anda, ketikkan perintah di bawah ini untuk membuat file konfigurasi yang berfungsi mengaktifkan fitur hibernate melalui antarmuka grafis.
    sudo nano /etc/polkit-1/localauthority/50-local.d/com.ubuntu.enable-hibernate.pkla

    _*ket :_Di sini saya menggunakan teks editor nano yang berbasis _CLI _supaya cepat, anda juga bisa menggunakan teks editor GUI seperti _gedit, geany, _dsb. dengan menyesuaikan perintah pada terminal tersebut.

  4. Kemudian copy dan paste isi file konfigurasi di bawah ini ke dalam file tersebut. Jika anda menggunakan _nano _seperti yang saya sarankan, tekan _Ctrl+O,_ lalu_ Enter, _kemudian_ Ctrl+X _untuk menyimpannya.

    **`[Re-enable hibernate by default in upower]
Identity=unix-user:*
Action=org.freedesktop.upower.hibernate
ResultActive=yes

[Re-enable hibernate by default in logind]
Identity=unix-user:*
Action=org.freedesktop.login1.hibernate;org.freedesktop.login1.hibernate-multiple-sessions
ResultActive=yes`**

  5. Terakhir restart komputer anda, lalu cek, pada menu tombol power panel unity pojok kanan atas layar akan muncul pilihan hibernate. Begitu juga pada prompt dialog yang muncul ketika kta menekan tombol power fisik komputer atau PC kita.

Gambar di bawah ini menunjukkan perubahan setelah menambahkan file konfigurasi di atas, pilihan hibernate muncul pada menu tombol power yang berada di pojok kanan atas layar, dan juga pada prompt dialog ketika kita menakan tombol fisik power pada PC atau komputer kita.

Saya ingatkan kembali, untuk memastikan bahwa kita sudah memiliki partisi swap pada PC atau komputer kita untuk digunakan. Selain sebagai cadangan RAM, partisi swap ini juga sangat bermanfaat untuk menyimpan data hibernate pada sistem operasi berbasis Linux secara umum, dan Ubuntu pada khususnya.
