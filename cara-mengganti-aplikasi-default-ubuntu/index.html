Beberapa dari kita mungkin menyukai berbagai aplikasi tersebut dan merasa aplikasi-aplikasi tersebut cukup untuk mengatasi atau membantu segala pekerjaan kita.

Namun, di sisi lain, beberapa dari kita juga merasa kurang nyaman dengan beberapa aplikasi bawaan tersebut. Misalnya saja ada orang yang lebih suka Google Chrome daripada Mozilla Firefox dan ingin mengganti web browser default-nya dengan Google Chrome. Nah, bagi beberapa orang, terutama pemula Ubuntu, mungkin masih merasa bingung tentang bagaimana cara mengganti aplikasi default Ubuntu.

Cara Mengganti Aplikasi Default Ubuntu :

  1. Klik pada icon ‚Äúpower‚Äù pada sudut kanan atas layar desktop Ubuntu anda.
  2. Kemudian klik pada pilihan ‚ÄúAbout This Computer‚Äù.
  3. Lalu akan muncul tampilan seperti dibawah, pada sidebar kanan, klik ‚ÄúDefault Application‚Äù.
  4. Langkah terakhir, pilih aplikasi default yang anda inginkan pada masing-masing kategori.

Aplikasi default yang terpilih pada setiap kategori akan meng-handle segala tugas yang berhubungan dengan jenis file atau tugas yang sesuai dengan kategori tersebut. Misalkan pada kategori video saya memilih VLC Media Player sebagai aplikasi default, maka setiap kali saya meng-klik dua kali pada segala format file video, file tersebut akan langsung dibuka menggunakan VLC Media Player. Namun, kita masih bisa menggunakan aplikasi lain yang bukan merupakan aplikasi default untuk membuka file tertentu dengan klik kanan, lalu memilih opsi ‚ÄúOpen With‚Äù seperti pada sistem operasi lain.
