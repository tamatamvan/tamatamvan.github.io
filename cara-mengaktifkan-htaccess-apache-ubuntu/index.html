Mengaktifkan .Htaccess Apache Pada Ubuntu

Posted on 2016-05-08 | Post modified 2017-06-21 | In Linux | 0 comments
Reading time 3

Assalamu‚Äôalaikum Wr. Wb., .htaccess (hypertext access) file adalah sebuah _directory-level configuration file, yang dapat digunakan pada beberapa jenis web server, salah satunya ialah Apache. File .htaccess sendiri biasa digunakan untuk berbagai keperluan seperti mengalihkan URL (URL Redirection), pemendekan URL (URL shortening), Access-security control (untuk beberapa halaman web dan file yang berbeda), dan berbegai penggunaan lainnya. Dalam artikel kali ini, kita akan membahas sedikit mengenai .htaccess, yaitu tentang cara mengaktifkan .htaccess_ Apache pada Ubuntu.

Dalam beberapa kasus, terkadang kita mencoba untuk menggunakan .htaccess untuk mengganti URL dari sebuah halaman web, mengalihkan URL, atau memendekkan URL, akan tetapi terkadang hal tersebut gagal dan memunculkan pesan error dari web server kita. Mungkin saja hal tersebut dikarenakan fitur .htaccess yang belum aktif pada web server yang kita gunakan.

Berikut ini saya menyusun artikel berisi tutorial cara mengaktifkan .htaccess Apache pada Ubuntu. Sebagai catatan, tutorial kali ini mungkin bisa saja anda terapkan pada distro Linux lain dengan menyesuaikan beberapa langkah yang ada sesuai dengan keperluan anda.

Cara Mengaktifkan .htaccess Apache pada Ubuntu

 

  1. Buka file _apache2.conf _dengan mengektikkan perintah di bawah pada terminal anda :

    **`sudo nano /etc/apache2/apache2.conf
`**

  2. Tekan tombol Ctrl+W, lalu masukkan ‚ÄúAccessFileName .htaccess‚Äú, apabila ada tanda pagar/comment (#) didepannya, hapus tanda itu.

  3. Kemudian temukan line atau baris dengan script seperti di bawah ini :

    **`<Directory /var/www/>
 Options Indexes FollowSymLinks
 AllowOverride None
 Require all granted
</Directory>
`**
    ganti "None" dengan "All" 
    **`AllowOverride All
`**

  4. Selanjutnya, coba check pada file 000-default.conf pada ‚Äò/etc/apache2/sites-available/000-default.conf', dengan memasukkan perintah :

    **`sudo nano /etc/apache2/sites-available/000-default.conf`**

  5. Kemudian lakukan hal yang sama seperti pada langkah ke tiga pada file _000-default.conf _tersebut.
    (ket : Langkah 4 dan 5 dilakukan tergantung pada versi Ubuntu yang anda gunakan, pada Ubutnu 16.04 seperti yang saya gunakan sendiri tidak perlu melakukan langkah nomor 4 dan 5 ).

  6. Selanjutnya aktifkan ModRewrite dengan mengetikkan perintah :

    **`sudo a2enmod rewrite`**

  7. Restart apache dengan perintah :

    **`sudo service apache2 restart`**

  8. Selesai, silakan coba gunakan .htaccess pada aplikasi web di web server anda.

Nah, itu dia tutorial mengenai cara mengaktifkan .htaccess Apache pada Ubuntu. Semoga tutorial kali ini dapat dengan mudah anda mengerti dan dapat bermanfaat bagi kita semua.
