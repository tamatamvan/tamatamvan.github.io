Membuat Launcher Aplikasi pada Ubuntu

Untuk membuat launcher aplikasi pada Ubuntu kita perlu sebuah aplikasi bernama Alacarte. Alacarte merupakan main menu editor yang dapat digunakan untuk menambah, menghapus, atau mengedit daftar aplikasi pada launcher distro Linux, salah satunya ialah Ubuntu.

Untuk menginstall Alacarte silakan jalankan perintah berikut pada terminal anda :

sudo apt-get install alacarte
