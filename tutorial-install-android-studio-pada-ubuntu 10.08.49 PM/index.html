Tutorial Install Android Studio pada Ubuntu Linux

Android Studio merupakan sebuah software IDE yang berjalan pada Oracle Java 1.8 atau lebih tinggi. Jadi pastikan sebelum menginstall Android Studio anda telah memiliki Oracle Java terinstall pada sistem anda. Jika belum, anda dapat mengikuti tutorial berikut untuk menginstall Oracle Java pada Ubuntu.

Oke, jika anda sudah memiliki Oracle Java pada sistem Ubuntu anda mari kita memulai untuk menginstall Android Studio. Ikuti langkah-langkah berikut.

  1. Download Android Studio untuk Linux di sini.
  2. Ekstrak file .zip yang telah anda download pada lokasi yang cocok yang anda inginkan, misalnya di /usr/local/ , untuk user profile anda, atau /opt/ untuk beberapa user. Atau anda bisa juga menempatkannya pada folder home anda.
  3. Selanjutnya, buka terminal Ubuntu anda, dan arahkan ke lokasi direktori android-studio/bin/, dan jalankan file studio.shdi dalamnya.
    Contoh: saya sendiri menempatkan file hasil ekstrak .zip android studio pada direktori /opt/maka saya akan menjalankan perintahcd /opt/android-studio/bin/ untuk masuk ke lokasi file android studio berada, dan ./studio.sh untuk menjalankan file studio.sh.
  4. Kemudian, pilih dan tentukan apakah anda ingin mengimport setting Android Studio yang sebelumnya atau tidak, jika anda baru pertama kali menginstall Android Studio, pilih tidak, laluk klik OK.
  5. Setelah itu akan muncul Android Studio Setup Wizard, ikuti setiap instruksinya. Yang pertama pilih tipe instalasi yang kita inginkan, Stadard atau Custom. Jika sudah silakan klik ‚ÄòNext‚Äô.
  6. Selanjutnya akan muncul tampilan yang meminta anda untuk memastikan atau memverifikasi konfigurasi yang anda inginkan. Jika semuanya OK, silakan klik ‚ÄòNext‚Äô.
  7. Selanjutnya, Android Studio Setup Wizard akan mendownload segala file yang dibutuhkan Android Studio untuk berjalan. Silakan bersabar menunggu hingga proses download selesai.
  8. Setelah proses download selesai, silakan klik finish. Untuk menjalankan Android Studio anda cukup melakukan langkah ketiga dari tutorial ini.

Nah, itu dia cara install Android Studio pada Ubuntu Linux. Pada tutorial ini setelah proses intallasi berhasil, kita perlu membuka terminal dan mengetikkan beberapa perintah untuk menjalankan Android Studio.
