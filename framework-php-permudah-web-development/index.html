Framework PHP Permudah Web Development

Posted on 2016-06-28 | Post modified 2017-06-21 | In Pemrograman , Pemrograman Web | 0 comments
Reading time 5

Assalamu‚Äôalaikum Wr. Wb. Saat ini begitu banyak web programmer atau developer yang menggunakan berbagai macam framework PHP untuk mempermudah dan mempercepat proses pembuatan website atau aplikasi web. Dari sekian banyak framework PHP yang ada sering kali membuat banyak pemula yang ingin mempelajari framework kebingungan menentukan pilihan. Oleh karena itu kali ini saya akan memberikan beberapa contoh framework PHP yang dapat anda coba untuk pelajari.

Framework sendiri merupakan sebuah software atau program yang merupakan kumpulan fungsi-fungsi, modul, prosedur, class-class yang siap untuk digunakan. Dengan menggunakan framework proses development suatu aplikasi dapat lebih cepat dan mudah.

Dibawah ini ada beberapa contoh framework PHP yang cukup populer untuk anda coba. Tidak perlu memperdebatkan framework mana yang lebih baik atau paling baik, karena pada dasarnya setiap framework memiliki kelebihan dan kekurangan masing-masing. Silakan untuk memilih framework yang anda rasa paling nyaman untuk digunakan.

Daftar Framework PHP

CodeIgniter





CodeIgniter PHP Framework

Codeigniter merupakan salah satu framework PHP yang saat ini digunakan oleh banyak web developer di Indonesia. Codeigniter memiliki performa yang cukup impresif dan mampu mengungguli beberapa PHP Framework pesaingnya. Selain itu CodeIgniter memiliki _small foot-print _atau ukuran file yang relatif kecil, serta cukup mudah dipelajari.

[Yii Framework

](http://www.yiiframework.com)

[caption id=‚Äùattachment_783‚Äù align=‚Äùaligncenter‚Äù width=‚Äù840‚Äù] Yii Framework[/caption]

YiiFramework dibangun berdasarkan teknologi web 2.0 dengan tingkat keamanan yang cukup tinggi. Yii Framework memiliki cukup banyak fitur menarik seperti Input Validation, output filtering, dan SQL Injection

Phalcon

[caption id=‚Äùattachment_781‚Äù align=‚Äùaligncenter‚Äù width=‚Äù840‚Äù] Phalcon PHP Framework[/caption]

Phalcon merupakan salah satu framework PHP dengan performa terbaik dan tercepat, dibangun dengan menggunakan pola konsep MVC seperti framework lainnya dan dibangun dengan bahasa pemrograman C, C++, dan PHP. Yang menjadikan framework ini menarik ialah framework ini merupakan sebuah C extension.

Laravel

[caption id=‚Äùattachment_779‚Äù align=‚Äùaligncenter‚Äù width=‚Äù840‚Äù] Laravel[/caption]

Laravel merupakan salah satu framework palng populer yang belakangan ini terus naik daun. Laravel sendiri memiliki berbagai fitur menarik seperti artisan, blade templating, eloquent, homestead, dan berbagai fitur menarik lainnya.

Panada

[caption id=‚Äùattachment_780‚Äù align=‚Äùaligncenter‚Äù width=‚Äù840‚Äù] Panada Framework[/caption]

Panada merupakan framework lokal Indonesia yang dikembangkan oleh Iskandar Soesman. Panada adalah high performance PHP development framework namun tetap sederhana. Tidak hanya sederhana dalam pengertian cara penggunaanya, tetapi juga bagaimana core system-nya bekerja. KISS atau Keep it Simple Son! selalu menjadi prinsip utama Panada dalam setiap pengembangan.

Helium

Helium juga merupakan framework MVC asal Indonesia yang dibangun oleh Andhika Nugraha (founder www.benangmerah.net) menggunakan bahasa pemrograman PHP.

Kecik Framework

[caption id=‚Äùattachment_778‚Äù align=‚Äùaligncenter‚Äù width=‚Äù840‚Äù] Kecik Framework[/caption]

Kecik Framework merupakan framework PHP sederhana asal Indonesia yang dapat anda bangun dan kembangkan untuk menjadi framwork yang lebih kompleks untuk anda gunakan. Kecik Framework dibangun dengan konsep MVC yang sederhana.

FuelPHP

[caption id=‚Äùattachment_777‚Äù align=‚Äùaligncenter‚Äù width=‚Äù840‚Äù] FuelPHP[/caption]

FuelPHP berkerja dengan konsep MVC dan didukung dengan adanya berbagai plug-in inovative. FuelPHP mendukung router based theory dimana anda dapat melakukan routing secara langsung menuju input url terdekat, membuat controller lebih dekat dan memberikan kontrol terhadap eksekusi selanjutnya.

Zend

[caption id=‚Äùattachment_784‚Äù align=‚Äùaligncenter‚Äù width=‚Äù840‚Äù] Zend Framework[/caption]

Zend merupakan framework PHP modern yang difokuskan untuk membangun aplikasi web tingkat tinggi. Zend bekerja berdasarkan Cryptographic and secure coding tools.

Flight PHP

[caption id=‚Äùattachment_776‚Äù align=‚Äùaligncenter‚Äù width=‚Äù840‚Äù] Flight[/caption]

Flight PHP merupakan framework yang sangat baik untuk mengembangkan aplikasi RESTful web service, dan menggunakan MIT Software License.

Cake PHP

[caption id=‚Äùattachment_774‚Äù align=‚Äùaligncenter‚Äù width=‚Äù840‚Äù] CakePHP[/caption]

Cake PHP merupakan framework PHP yang bagus untuk mengembangkan aplkasi web yang sederhana dan bagus.CakePHP memiliki berbagai fitur bawaan menarik seperti validation, SQL injection prevention yang membuat aplikasi web anda lebih aman. Keunggulan CakePHP antara lain : dibangun cepat, tidak memperlukan konfigurasi, menggunakan MIT license, MVC, dan aman.

PHPixie

[caption id=‚Äùattachment_782‚Äù align=‚Äùaligncenter‚Äù width=‚Äù840‚Äù] PHPixie Framework[/caption]

PHPixie berkerja dengan konsep MVC, dan didesign untuk membangun aplikasi web dengan cepat dan mudah.

 

Nah, itu dia daftar dari beberapa framework PHP yang mungkin dapat menjadi referensi bagi anda untuk menentukan framework yang akan anda pelajari. Sekali lagi, setiap framework memiliki kelebihan dan kekurangan masing-masing. Cukup pilih yang paling nayaman untuk anda gunakan dan sesuai dengan kebutuhan anda.

Semoga informasi ini dapat bermanfaat bagi kita semua, apabila anda memiliki tambahan, pertanyaan, kritik, atau saran, silakan sampaikan melalui kolom komentar dibawah ini.
