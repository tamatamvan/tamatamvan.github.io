Linux sendiri banyak digunakan sebagai sistem operasi komputer server, desktop, atau bahkan sistem operasi mobile seperti Android dan Ubuntu Phone. Sistem Linux sendiri tentu berbeda dengan sistem operasi yang banyak digunakan seperti Windows. Jika anda ingin mempelajari Linux dengan mudah dan menyenangkan, sekarang anda dapat belajar linux melalui situs Linux Journey.

Linux Journey sendiri merupakan sebuah website yang dibangun oleh pengguna forum Reddit bernama WarbyDarby, dan didesign untuk membantu anda belajar bagaimana cara Linux bekerja.

Situs ini tidak menggunakan cara belajar tradisional dengan melalui artikel yang klise. Situs ini lebih fokus untuk memberikan pelajaran mengenai apa yang penting dari sistem Linux itu sendiri. Dengan belajar di situs Linux Journey ini sendiri anda akan belajar bagaimana bagian-bagian penting dari Linux tersebut bekerja (tidak tergantung distro apa yang anda gunakan, dan bersifat Linux secara umum), bagaimana anda bisa memanfaatkannya, menguasainya dan/atau belajar dari hal tersebut.

Anda dapat memilih materi untuk dipelajari tidak peduli anda mau belajar Linux mulai darimana. Entah anda ingin belajar vim atau emacs, mempelajari lebih banyak tentang Linux, atau belajar mengenai penggunaan praktikal tool pada Linux, anda tetap dapat memilihnya. Anda bisa mengunjungi situs Linux Journey sekarang juga, dan memberikan masukkan atau juga pujian untuk pembuatnya melalui Reddit.

Baiklah, itu dia sekilas mengenai situs Linux Journey, sekarang saya akan merangkum setiap kelebihan dan kekurangan dari situs Linux Journey sebagai berikut :

Kelebihan :

  • Materi pembelajaran cukup lengkap dan dikelompokkan secara rapi.
  • Penyampaian materi to the point dan cukup jelas sehingga mudah dipahami.
  • Bersifat open source sehingga setiap orang dapat berkontribusi.
  • Terdapat latihan dan kuis dalam setiap bab.
  • Design menawan.

Kekurangan :

  • Belum tersedia dalam Bahasa Indonesia.
  • Belum ada konsol interaktif.

Bagi anda yang ingin belajar Linux serta memiliki kemampuan berbahasa Inggris yang cukup baik, maka situs Linux Journey ini patut untuk anda coba.
