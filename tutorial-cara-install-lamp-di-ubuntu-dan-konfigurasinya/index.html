Bagi seorang web programmer, kehadiran sebuah local web server pasti sangatlah dibutuhkan. Local web server ini nantinya akan digunakan untuk menguji coba apakah aplikasi berbasis web yang kita buat apakah sudah berjalan dengan baik atau belum.

Nah, kali ini saya akan berbagi mengenai cara install dan konfigurasi LAMP (Linux, Apache, MySQL, PHP, phpmyadmin) di Ubuntu.

 

1. Cara Install LAMP di Ubuntu

Hal pertama yang perlu kita lakukan adalah menginstall LAMP itu sendiri. Berikut ini adalah cara install **LAMP di Ubuntu :**

  1. Buka terminal, ketikkan perintah :

    sudo apt-get install apache2 mysql-server php5 phpmyadmin`

  2. Selanjutnya masukkan password user anda, dan tekan enter. Tunggu proses download dan instalasi.
  3. Pada saat proses instalasi, akan muncul perintah untuk mengonfigurasi phpmyadmin seperti pada gambar dibawah. Tekan tombol ‚ÄòTab‚Äô untuk memilih ‚ÄòYes‚Äô, lalu tekan ‚ÄòEnter‚Äô.
  4. Selanjutnya masukkan MySQL application password untuk phpmyadmin seperti pada gambar dibawah ini sesuai dengan keinginan anda.Tekan ‚ÄòTab‚Äô untuk memilih ‚ÄòOk‚Äô, lalu tekan ‚ÄòEnter‚Äô
  5. Selanjutnya lakukan konfirmasi password dengan memasukkan password yang sama dengan yang anda masukka sebelumnya. Tekan ‚ÄòTab‚Äô untuk memilih ‚ÄòOk‚Äô, lalu tekan ‚ÄòEnter‚Äô.
  6. Langkah berikutnya, masukkan password untuk database phpmyadmin anda. Password inilah yang akan anda gunakan untuk mengakses phpmyadmin nantinya. Tekan ‚ÄòTab‚Äô untuk memilih ‚ÄòOk‚Äô, lalu tekan ‚ÄòEnter‚Äô.
  7. Selanjunya pilih web server yang akan digunakan untuk menjalankan phpmyadmin, tekan ‚Äòspasi‚Äô untuk memilih apache2, Tekan ‚ÄòTab‚Äô untuk memilih ‚ÄòOk‚Äô, lalu tekan ‚ÄòEnter‚Äô.

  8. Selanjutnya tunggu hingga proses instalasi selesai.

    2. Konfigurasi Direktori public_html pada Apache2

    Setelah instalasi selesai sebenarnya LAMP web server di Ubuntu kita sudah bisa digunakan. Namun disini saya akan menambahkan beberapa konfigurasi untuk memudahkan perkerjaan kita kedepannya. :)

  9. Buatlah folder/direktori public_htmlpada direktori home. Untuk hal ini kita bisa menggunakan perintah mkdir /home/username_anda/public_html/,atau langsung menggunakan file manager nautilus atau yang lainnya. Folder/direktori ini nantinya akan menjadi tempat penyimpanan semua file project yang berupa website atau aplikasi berbasi web milik kita.

  10. Ubah permissions pada folder public_htmlyang telah kita buat dengan mengetikkan perintah sudo chmod 777 /home/username_anda/public_html/pada terminal, atau menggunakan file manajer nautilus dengan cara _klik kanan pada folder publichtml ‚Äì> Properties ‚Äì> Permissions ‚Äì> Ubah semua hak akses menjadi ‚ÄòCreate and delete files‚Äô ‚Äì> klik ‚ÄòChange Pemissions for Enclosed Files‚Äô.

  11. Selanjutnya buat link menuju folder/direktori public_htmlpada folder/direktori /var/www/html/, dengan mengetikkan perintah pada terminal :

    `sudo ln -s /home/username_anda/public_html/ /var/www/html/public_html/`

    3. Pengujian Instalasi LAMP pada Ubuntu

    Setelah proses instalasi selesai, selanjutnya kita lakukan pengujian untuk memeriksa apakah hasil instalasi dan konfigurasi LAMP pada Ubuntu milik kita telah berhasil dan dapat berjalan dengan baik.

  12. Langkah pertama, buka web browser anda. Lalu ketikkan ‚Äúlocalhost‚Äù atau ‚Äúhttp://localhost‚Äú pada address bar di browser anda, lalu tekan enter. Jika browser anda menampilkan halaman web seperti pada gambar dibawah ini, itu berarti web server apache2 telah berjalan dengan baik di Ubuntu kita.
    [caption id=‚Äùattachment_112‚Äù align=‚Äùaligncenter‚Äù width=‚Äù1024‚Äù] Halaman default apache2 pada Ubuntu[/caption]
     

  13. Selanjutnya ketikkan alamat ‚Äúlocalhost/public_html‚Äù atau ‚Äúhttp://localhost/public_html‚Äú pada address bar browser anda. Jika konfigurasi direktori public_html berhasil, akan muncul halaman web bertuliskan ‚ÄúIndex of public_html‚Äù pada browser anda seperti pada gambar dibawah.

  14. Langkah selanjutnya adalah memeriksa instalasi PHP. Buka terminal anda, kemudian ketikkan perintah :

    sudo nano /var/www/html/info.php

    Lalu, pada editor nano masukkan script php di bawah ini :
    `<?php
     phpinfo();``
     ?>

    Kemudian tekan ‚ÄòCtrl+O‚Äô, lalu ‚ÄòEnter‚Äô untuk menyimpan file info.php. Kemudian ‚ÄòCtrl+X‚Äô untuk keluar dari nano.

  15. Akses file info.phpdengan mengetikkan ‚Äòlocalhost/info.php‚Äô pada address bar di browser anda. Maka akan muncul tampilan seperti dibawah ini.
  16. Terakhir, kita uji mysql-server dan phpmyadmin yang telah kita install dan konfigurasi, dengan mengetikkan alamat ‚Äúlocalhost/phpmyadmin‚Äù atau ‚Äúhttp://localhost/phpmyadmin‚Äú. Jika muncul tampilan seperti dibawah, masukkan ‚Äúroot‚Äù sebagai username, dan password database phpmyadmin yang telah anda isikan di konfigurasi phpmyadmin pada saat instalasi sebagai passwordnya, lalu klik ‚ÄúGo‚Äù.
  17. Jika muncul tampilan seperti gambaw dibawah itu berarti mysql-server dan phpmyadmin telah berjalan dengan baik. Serta Instalasi dan Konfigurasi LAMP di Ubuntu secara keseluruhan telah berhasil dilakukan.
